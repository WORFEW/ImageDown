# ImageDown

一个轻量的浏览器扩展，用于批量抓取并下载网页中的图片，支持将图片打包为 ZIP 文件并一次性保存到本地。适用于需要批量保存网页图片的场景(例如图库、商品图、博客图集等)。

**主要功能**
- 一键扫描当前页面上的图片资源
- 双击可以放大查看图片
- 选择性下载(选中要下载的图片或全部下载)
- 图片打包为 ZIP 下载(基于 `jszip`)
- 导出图片链接(复制到剪切板)

**安装（开发者 / 本地调试）**
1. 在浏览器中打开扩展管理页面：`chrome://extensions` 或 `edge://extensions`。
2. 启用右上角的“开发者模式”。
3. 点击“加载已解压的扩展程序(Load unpacked)”，选择本项目目录(包含 `manifest.json` 的文件夹)。
4. 安装后会在工具栏出现扩展图标，点击即可使用。

**使用说明**
- 点击工具栏扩展图标打开弹窗，点击“扫描”或“开始”按钮，扩展会列出页面中发现的图片。
- 在弹窗中勾选要下载的图片，选择“下载所选”或“下载全部”。
- 扩展会将所选图片打包为 ZIP 并触发下载
